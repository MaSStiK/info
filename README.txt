Создает копию в папку info:
git clone https://github.com/MaSStiK/info.git info

Переход в деректорию выше:
cd info

Добавление файлов в коммит:
git add .

Запись коммита:
git commit -m "Комментарий"

Отправить коммит на сервер:
git push

Принять изменения:
git pull

Авторство коммита:
git --global user.email "Ваш емаил"
git --global user.name "Ваше имя"
